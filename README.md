## Before You Open An Issue

- Use `Bug Report` for something broken or not working as expected.
- Use `Feature Request` for ideas and improvements.
- Use `Support Request` if you need help, setup guidance, or troubleshooting.
- Join the `Stream Havoc Discord` for community support and discussion.
